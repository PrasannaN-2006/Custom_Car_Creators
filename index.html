Chapter I

About the Project

Custom Car Creators is an innovative web-based platform for car enthusiasts who want to design and build their own customized vehicles. The system provides a simple and interactive interface for selecting car models and modifying features like color, engine type, wheels, interior design, and accessories.

The platform allows users to visualize their customized car design before finalizing choices — no advanced technical knowledge needed.

It also streamlines the vehicle customization process by managing customer selections, storing design data, and generating customized car configurations efficiently.

Custom Car Creators is beneficial for car enthusiasts, collectors, and automotive professionals who want to explore custom car designs in a modern, digital platform.

Chapter II — Proposed System

Key Features

Everything the system offers — from user auth to admin dashboards — designed to replace slow, manual spare-parts workflows.

🔐

User Auth

Secure registration and login with role-based access control for admins and regular users.

🔍

Product Browsing

Browse different categories of car parts — tyres, suspension, lighting, exhaust systems — and view detailed product info.

🛒

Shopping Cart

Add products, update quantities, remove items, and see total price before checkout — all in one place.

📦

Order Management

Place orders online with a simple checkout. Track order status (Pending / Delivered) and view full order history.

🛠

Admin Dashboard

Administrators can manage users, products, categories, and orders. View sales stats and monitor the whole system.

📂

Category System

Organize products into categories, filter by type, and make browsing faster for customers.

Chapter IV — Testing

Testing Strategy

Multiple levels of testing were carried out to make sure everything works correctly and securely.

✅ Functional Testing

Unit, integration, and end-to-end testing covering login, product browsing, shopping cart, and order placement.

⚡ Performance Testing

Load testing, stress testing, and response time checks for product listings, cart pages, and admin dashboards.

🔒 Security Testing

SQL injection protection, vulnerability scanning, and authentication checks to secure user data and admin access.

📱 UX Testing

Usability, mobile responsiveness, and accessibility testing to ensure a smooth experience on all devices.

🌍 Compatibility Testing

Verified across Chrome, Firefox, Edge, and Safari. Also tested on desktop, laptop, tablet, and mobile.

🔁 Regression Testing

Manual and automated regression testing after each update to make sure changes don't break existing features.

Chapter VI

Conclusion

The Custom Car Creators – Car Parts Selling System was developed to simplify and automate the process of selling automobile spare parts through an online platform. Traditional stores suffer from data loss, manual errors, time delays, and limited reach — this system addresses all of that.

Customers can easily browse car parts by category, compare products, and place orders online without visiting a physical store. Administrators get a full dashboard for managing products, users, and orders efficiently.

Built on Python Flask and SQLite, the system is reliable, secure, and ready for multiple users. With future enhancements like payment integration, order tracking, and mobile support, this platform can evolve into a fully-featured automobile e-commerce solution.
